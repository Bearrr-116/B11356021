# 6 個修正版流程圖（Mermaid）

以下為你先前無法渲染的 6 張流程圖，已改成較穩定的 Mermaid 寫法（節點文字使用雙引號）。

---

## 4) 會員－提出退款申請
```mermaid
flowchart TD
    A["開始"] --> B["會員登入系統"]
    B --> C{"登入成功?"}
    C -- 否 --> D["顯示登入錯誤"] --> Z["結束"]
    C -- 是 --> E["點選 提出退款申請"]
    E --> F["選擇課程與退款原因"]
    F --> G["上傳佐證資料（選填）"]
    G --> H["送出申請"]
    H --> I["系統驗證資料完整性"]
    I --> J{"驗證通過?"}
    J -- 否 --> K["提示補件或修正資料"] --> Z
    J -- 是 --> L["建立退款申請單"]
    L --> M["狀態設為 審核中"]
    M --> N["通知管理員待審核"]
    N --> Z
```

## 5) 會員－查看退款狀態
```mermaid
flowchart TD
    A["開始"] --> B["會員登入系統"]
    B --> C{"登入成功?"}
    C -- 否 --> D["顯示登入錯誤"] --> Z["結束"]

    E --> F["系統查詢退款申請紀錄"]
    F --> G["顯示狀態：審核中、核准、拒絕、已退款"]
    G --> H{"有更新?"}
    H -- 是 --> I["顯示最新處理時間與結果說明"]
    H -- 否 --> Z
    I --> Z
```

## 9) 教職員－更新課程紀錄

